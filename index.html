obdi-dev-repository

Repository metadata store for Obdi system plugins.

Add https://github.com/mclarkson/obdi-dev-repository.git to Repository URLs in the Obdi Admin interface.

Further documentation is available from the 'view' links below.

Contains

Name Description URL
helloworld The base of all helloworld plugins that creates a top level menu entry. view
helloworld-minimal Basic plugin with buttons and a modal dialog box. view
helloworld-runscript Plugin to demonstrate running a script on an Obdi worker. view
helloworld-localdb Plugin to demonstrate CRUD on a private db. view
